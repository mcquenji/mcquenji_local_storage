LocalStorageModule

LocalStorageModule class

Provides local storage access in a modular way.

You need to provide an LocalStorageSerializer instance for the type you want to store by calling InjectorUtils.addSerde.

Example:

@override
void binds(Injector i) {
   // This is required to store MyModel instances.
   i.addSerde<MyModel>(
     fromJson: MyModel.fromJson,
     toJson: (m) => m.toJson(),
   );
}
